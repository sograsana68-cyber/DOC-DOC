# DOC-DOC

Family document organizer: React + Axios frontend, Express + JWT backend.

## Run locally

**Backend** (port 3001):

```bash
cd backend && npm install && npm run dev
```

**Frontend** (port 5173):

```bash
cd frontend && npm install && npm run dev
```

Demo login: `demo@example.com` / `password123`

Set `VITE_API_URL` in `frontend` if the API is not at `http://localhost:3001`.
